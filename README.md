# yj_imap_attachment_downloader
Downloads all pdf files attech to mails in a given IMAP folder
## requirements
- python3
- imaplib

## usage
This is not very sofisticated. You fill in your credentials in the python script, run it and get a folder with all your files.
I made this mainly to download all tickets from Deutsche Bahn I used to keep archived in my mailbox for years.
